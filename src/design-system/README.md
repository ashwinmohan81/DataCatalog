# Design System

- **Tokens:** `tokens.css` – colors, typography, spacing, layout (sidebar width, topbar height, radius).
- **Layout:** App shell = top bar (nav + search + notifications + persona) + sidebar (domain tree) + main content. Max content width `var(--content-max)`.
- **Components:** Use tokens for all shared UI (cards, tables, filters, badges, tabs). Marketplace = same components with curated/consumer emphasis.
